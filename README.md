# React Projects & Experience  

I specialize in building modern, responsive, and interactive web applications with **React**. My work ranges from UI components to full-scale applications with state management, API integrations, and routing.  

## React Hooks & State Management  
- Built **dynamic counters, toggles, and reminders** using `useState`.  
- Implemented **data fetching & conditional rendering** with `useEffect` (Tours App, Reviews App).  
- Used `useRef` for DOM interactions and handling persisted values.  
- Managed **global state** with `useContext` and advanced workflows using `useReducer`.  

## Interactive UI Components  
- Developed **responsive navigation bars**, menus, and tab systems.  
- Created **sliders & carousels** for image galleries and testimonials.  
- Designed **forms with validation** for smooth user input handling.  
- Built reusable **modals and sidebars** with context-based state management.  

## Tools & Generators  
- **Lorem Ipsum Generator** for placeholder text.  
- **Color Shades Generator** for dynamic color palettes.  
- **Grocery Bud App** – A task & shopping list manager with CRUD features.  

## Advanced Applications  
- **Stripe Submenu Project** — complex menu architecture with context/state.  
- **Cart Project** — shopping cart with reducers, context, and state optimization.  
- **Cocktails App** — integrated external API with React Router, context, and dynamic filtering.  

## Routing & Performance  
- Implemented **multi-page navigation** with React Router.  
- Optimized performance with `useCallback` and memoization techniques.
